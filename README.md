# Guqula

Guqula is a smart image optimization web application that converts and compresses images into the most efficient format while preserving quality.

## Features

- Multi-format support (JPG, PNG, WebP)
- Intelligent format selection (AUTO mode)
- Batch image processing
- ZIP download for multiple files
- Real-time compression statistics

## Tech Stack

- FastAPI (Backend)
- Netlify (Frontend)
- Render (API Hosting)
